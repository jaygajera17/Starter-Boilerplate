# Starter-Boilerplate

Starter setup for multiple stacks so you can start building faster. Each stack lives in its own folder with its own README and setup steps.

## Stacks included

| Name                             | Tech stack                                           |
| -------------------------------- | ---------------------------------------------------- |
| [Node-Auth](Node-Auth/README.md) | TypeScript, Node.js, Express, MongoDB, Redis ,OAuth2 |

## Getting started

1. Pick a stack folder.
2. Read its README for setup, env vars, and run commands.

## Repo structure

```
Starter-Boilerplate/
	Node-Auth/
```
